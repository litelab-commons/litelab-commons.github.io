## ライセンス

公開中の教材はMIT Licenseです。改変・再配布・商用利用すべて自由です。

## 使い方

各教材はHTMLファイル単体で動作します。サーバー不要。
ダウンロードしてブラウザで開くだけで使えます。

## 投稿・貢献

あなたが作ったIT教材もここに置けます。
Pull Requestまたは[教材掲載リクエスト](https://github.com/litelab-commons/litelab-commons.github.io/issues/new?template=material-request.yml)でご連絡ください。

### 掲載条件

掲載する教材は、以下の条件を満たしていることを確認します。

1. ITリテラシー向上に役立つこと
2. 教育・研修・啓発目的で使えること
3. ライセンスが明確であること
4. 商用利用・改変・再配布の可否が明記されていること
5. ブラウザ教材は、ブラウザ上で安全に試せること
6. 個人情報・機密情報を外部送信しないこと
7. 危険な操作や攻撃手法を実行可能な形で提供しないこと
8. 配布プログラムは、実行内容・外部通信・必要権限・対応OSが明記されていること

掲載は、ライセンス・安全性・教育目的を確認したうえで行います。
まずはIssueで教材の概要、対象者、利用条件、デモURLまたはリポジトリURLを共有してください。
