詐欺サイト判定くん

サポート詐欺・偽セキュリティ警告ページの特徴を、ブラウザ内でローカル判定する Chrome / Edge 向け研修・検証用拡張機能です。 危険らしい特徴を検出すると、全画面オーバーレイで警告します。

免責事項

  • この拡張機能は研修・検証用です。完成済みの一般向けセキュリティ製品ではありません。
  • すべての詐欺サイトを検出できるものではありません。検出漏れがあります。
  • 正規サイトを誤って警告する可能性があります。警告は悪質性の断定ではありません。
  • 警告が出ないことは、そのページの安全性を保証しません。
  • 実在する詐欺サイトへアクセスして検証することは推奨しません。同梱の安全なテストページを使ってください。
  • 利用、改変、配布、判定結果に基づく判断は利用者自身の責任で行ってください。

権限とプライバシー

  • 判定のため、拡張機能はすべてのWebページでコンテンツスクリプトを実行します。
  • 現バージョンでは外部API・外部サーバーへの通信は行いません。
  • ページ内容、閲覧履歴、判定結果を開発者へ送信しません。
  • ホワイトリストのみ、利用者のブラウザ内の chrome.storage.local に保存します。
  • 詳細は ZIP 内の PRIVACY.md と DISCLAIMER.md を確認してください。

含まれるもの

  • プライバシーポリシーと免責事項

使い方

  1. ZIPを展開します。
  2. chrome://extensions または edge://extensions を開きます。
  3. デベロッパーモードをONにします。
  4. 「パッケージ化されていない拡張機能を読み込む」から scam-detector フォルダを選択します。
  5. 動作確認には testing/test-scam-page.html を使います。
