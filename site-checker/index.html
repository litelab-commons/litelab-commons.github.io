Browser Extension / Site Review Aid

Site Checker: 不審サイト確認ミニツール

表示中サイトのドメイン、DNS、RDAP、外部リソース、メールドメインを確認し、 要確認リストや全画面ブロックで詐欺サイト確認を補助する Chrome / Microsoft Edge 用拡張機能です。 安全・危険を断定するツールではなく、確認観点を整理するための補助ツールです。

Version 0.2.0 Chrome / Edge Manifest V3 MIT License
ZIPをダウンロード GitHubで見る LiteLab Commonsへ戻る

できること

  • 表示中ページのURL、ホスト名、登録ドメインを確認
  • Google Public DNS over HTTPSでA/AAAAレコードを確認
  • RDAPで登録日、レジストラ、ドメイン年齢を確認
  • ページ内の外部リソースドメインとメールドメインを確認
  • 信頼URL・信頼ドメイン・要確認URL・要確認ドメインを設定
  • ページ側からの全画面表示要求を既定でブロック
  • 確認履歴をブラウザ内に保存し、JSON/CSVで書き出し
これは確認補助ツールです。最終判断は、公式サイト、契約情報、決済方法、連絡先、証明書、組織情報など複数の材料で確認してください。

インストール

  1. ZIPをダウンロードして展開します。
  2. chrome://extensions または edge://extensions を開きます。
  3. デベロッパーモードを有効にします。
  4. 「パッケージ化されていない拡張機能を読み込む」を選びます。
  5. 展開した site-checker フォルダを選択します。

Chrome Web Store配布ではなく、研修・検証向けの手動インストール版です。

必要な権限

  • activeTab: 現在開いているページを確認するため
  • scripting: ページ内のリンク、リソース、メールアドレスを確認するため
  • tabs: 現在タブのURL取得と履歴ページ表示のため
  • storage: 設定と確認履歴をブラウザ内に保存するため
  • http://*/* / https://*/*: Webページ上で警告表示と全画面ブロックを動かすため
ブラウザ拡張としては強い権限を使います。中身を確認したうえで、研修・検証用途で利用してください。

データと外部通信

設定と履歴は chrome.storage.local に保存されます。開発者のサーバーへ利用履歴や閲覧履歴を送る処理は含めていません。

ただし、確認のために以下の外部サービスへ対象ドメイン名またはホスト名を問い合わせます。

  • dns.google: DNS A/AAAAレコード確認
  • rdap.org: RDAPドメイン登録情報確認
