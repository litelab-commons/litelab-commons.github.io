できること

  • ホスト画面をプロジェクターに表示し、複数人で参加できます。
  • 参加者はブラウザから名前を入れて参加し、各ラウンドでサイコロを振ります。
  • ランサムウェアやフィッシングなどのイベントで、ptや位置が変化します。
  • 防御アイコンにより、バックアップ、MFA、EDR、CSIRT、BCPの効果を体感できます。
  • 最終結果では、着順とSECURITY PTのランキングを表示します。

実行内容

  • FastAPIサーバーを起動し、ホスト画面と参加者画面を配信します。
  • WebSocketで参加者の状態、サイコロ、ゲーム進行を同期します。
  • 同一LAN内の端末からアクセスできるよう、ポート 8000 で待ち受けます。
  • ホスト画面のQRコード生成には、オンライン時のみCDNライブラリを利用します。読み込めない場合も参加URLは表示されます。

起動方法

  1. Python 3.10以降を用意します。
  2. このフォルダ一式をダウンロードします。
  3. Windowsでは start.bat をダブルクリックします。
  4. ホストPCで http://localhost:8000/ を開き、プロジェクターに表示します。
  5. 参加者は同じLAN内から http://<ホストPCのIP>:8000/play にアクセスします。
python -m pip install -r requirements.txt
python server.py

ネットワーク要件

  • ホストPCと参加者端末が同一LANにいる必要があります。
  • Windowsファイアウォールでポート 8000 の受信を許可してください。
  • 公共Wi-Fiや不特定多数が接続できるネットワークでは使わないでください。

同梱ファイル

  • server.py - サーバーとゲームロジック
  • host.html - ホスト画面
  • client.html - 参加者画面
  • requirements.txt - Python依存関係
  • README.md - 詳細な使い方
