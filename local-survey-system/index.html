ローカルアンケート集計システム

研修会場や授業内のローカルネットワークで、参加者から回答を集め、その場で集計表示できる Node.js 製のミニツールです。 管理画面、回答画面、スライド形式の結果表示をまとめて使えます。

利用前の注意

  • このツールはローカル研修用です。インターネットへ公開しないでください。
  • 管理画面に認証はありません。同じネットワーク上の人は管理画面へアクセスできます。
  • 回答データはメモリ保存です。サーバーを終了すると回答は消えます。
  • 個人情報や機微情報を集める用途には使わないでください。

できること

  • 設問を作成して受付開始/停止を切り替え
  • 単一選択、複数選択、5段階評価、自由記述に対応
  • 回答数や選択肢別の集計を管理画面で確認
  • 結果表示画面を投影してリアルタイム共有
  • アンケートと回答をローカルファイルに保存し、再起動後も復元
  • アンケートプリセットを最大5件保存
  • 回答結果をテキストファイルとして保存
  • アンケート定義を JSON でインポート/エクスポート

使い方

  1. ZIPを展開します。
  2. フォルダ内で npm install を実行します。
  3. npm start で起動します。
  4. http://localhost:3001/admin で管理画面を開きます。
  5. 参加者へ http://<PCのIP>:3001/ を共有します。
