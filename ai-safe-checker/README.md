# AIセーフ・チェッカー 🛡️

> 生成AIに入力してOK？それともNG？  
> 企業・学校のAIリテラシー研修で使える、ブラウザ完結型の判定ゲームです。

---

## 概要

「AIセーフ・チェッカー」は、生成AIへの情報入力リスクを直感的に学べるインタラクティブな研修ツールです。  
**1ファイル（HTML）で動作**し、サーバー不要・インストール不要で即利用できます。

### 特徴

- **30問プールからランダム10問を出題** — 繰り返しても飽きない設計
- **OK / NG / 条件付きOK（MASK）** の3分類で現実に即した判定
- **危険キーワードのハイライト表示** — どこがリスクかが一目でわかる
- **カテゴリ別弱点診断** — 個人情報・機密情報・著作権などの苦手分野が可視化
- **「安全な使い方」の提示** — NGで終わらず、正しい使い方を提案
- **スマホ・PC両対応** のレスポンシブUI

## カスタマイズ方法

コード冒頭の `POOL` 配列を編集するだけで、自社のセキュリティ規約に合わせたオリジナル問題を追加できます。

### 問題データの構造

```javascript
{
  text: "問題文をここに書く（実際にありそうなシチュエーション）",
  answer: "NG",           // "OK" / "NG" / "MASK"（条件付きOK）
  riskLevel: 5,           // 2=低 / 3=中 / 4=中高 / 5=高
  category: "機密情報",   // 自由に設定可能
  explanation: "なぜNGなのかの解説文",
  safeAction: "どうすれば安全に使えるかの提案（NGの場合に表示）",
  highlights: ["危険キーワード1", "危険キーワード2"]  // 問題文中でハイライトする語句
}
```

### answer の種類

| 値 | 意味 | 画面上の扱い |
|---|---|---|
| `"OK"` | 入力して問題なし | OKボタンが正解 |
| `"NG"` | 入力してはいけない | NGボタンが正解 |
| `"MASK"` | 匿名化・加工すればOK | NGボタンが正解（解説で条件を提示） |

### カスタム例（自社規約向け）

```javascript
{
  text: "社内の〇〇システムのログイン情報をAIに入力して手順書を作ってもらった",
  answer: "NG",
  riskLevel: 5,
  category: "社内情報",
  explanation: "認証情報は絶対に外部サービスに入力してはいけません。",
  safeAction: "ログイン情報を伏せた状態で手順のみをAIに相談しましょう。",
  highlights: ["ログイン情報"]
}
```
